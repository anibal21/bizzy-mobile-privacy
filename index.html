🔒 Política de Privacidad - Bizzy Mobile

📅 Fecha de última actualización: 23 de abril de 2026
📱 Aplicación: Bizzy Mobile
🏢 Desarrollador: Bizzy

1. 📷 Información que Recopilamos

1.1 Datos de la Cámara

  • Propósito: Permitir a los usuarios tomar fotografías dentro de la aplicación
  • Uso: Las fotografías se procesan localmente en el dispositivo
  • Almacenamiento: Las imágenes se almacenan temporalmente para su procesamiento
  • Compartición: No compartimos las imágenes con terceros sin consentimiento explícito

1.2 Acceso a Archivos Multimedia

  • Propósito: Permitir a los usuarios seleccionar imágenes de su galería
  • Uso: Solo accedemos a las imágenes específicamente seleccionadas por el usuario
  • Limitaciones: No escaneamos ni accedemos a toda la galería

1.3 Datos Biométricos

  • Propósito: Autenticación segura del usuario (huella dactilar, Face ID)
  • Uso: Solo para verificar la identidad del usuario al acceder a la aplicación
  • Almacenamiento: Los datos biométricos no se almacenan en nuestros servidores
  • Procesamiento: Se procesan localmente en el dispositivo del usuario

2. 🔐 Cómo Usamos la Información

2.1 Funcionalidad de la Aplicación

  • Permitir la captura y selección de imágenes
  • Proporcionar autenticación segura
  • Mejorar la experiencia de usuario con retroalimentación háptica
🛡️ Importante: Los datos biométricos permanecen en el dispositivo del usuario y no se envían a nuestros servidores.

3. 🚫 Compartición de Información

3.1 Con Terceros

  • No compartimos datos de la cámara, biométricos o de archivos multimedia con terceros
  • Las imágenes solo se procesan para la funcionalidad específica de la aplicación
  • No vendemos ni alquilamos información personal

4. 💾 Almacenamiento y Seguridad

4.1 Almacenamiento Local

  • Los datos se almacenan de forma segura en el dispositivo del usuario
  • Utilizamos las mejores prácticas de seguridad móvil
  • Los datos se cifran cuando es posible

4.2 Retención de Datos

  • Las imágenes temporales se eliminan después del procesamiento
  • Los datos biométricos permanecen solo en el sistema seguro del dispositivo
  • No mantenemos copias de datos sensibles en nuestros servidores

5. ⚙️ Derechos del Usuario

5.1 Control de Permisos

  • Los usuarios pueden revocar permisos en cualquier momento desde la configuración del dispositivo
  • La aplicación solicitará permisos solo cuando sea necesario para la funcionalidad

5.2 Eliminación de Datos

  • Los usuarios pueden eliminar datos de la aplicación desde la configuración del dispositivo
  • Los datos temporales se eliminan automáticamente

6. 📝 Cambios en la Política

Nos reservamos el derecho de actualizar esta política de privacidad. Los cambios significativos se notificarán a través de la aplicación.
